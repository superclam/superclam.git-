📋 使用说明

  1. 下载并安装 ScreenHelper APK 文件
  2. 首次启动时授予应用所需的权限（存储、悬浮窗等）
  3. 打开应用，启用"检测绕过模式"
  4. 点击"启动服务"并授予屏幕录制权限
  5. 打开目标应用（如学习通）
  6. 正常使用截屏功能，检测将被绕过

🔧 技术原理

本应用基于虚拟机环境中检测失效的原理开发。在虚拟机内运行目标应用时， 截屏和分屏检测会失效，因为实际的截屏操作发生在宿主机而非虚拟机内。

ScreenHelper 通过技术手段模拟这种环境特征，让检测机制认为应用运行在虚拟环境中， 从而实现绕过检测的目的。
